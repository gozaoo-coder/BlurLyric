# BlurLyric Tauri 安卓版本适配 - 产品需求文档

## 概述
- **Summary**: 为现有的 BlurLyric 音乐播放器应用适配 Tauri 框架的安卓平台支持，使应用能够在安卓设备上运行，保持与桌面版本一致的功能和用户体验。
- **Purpose**: 扩展 BlurLyric 的适用平台范围，让用户能够在安卓设备上使用这款音乐播放器，享受统一的跨平台体验。
- **Target Users**: 希望在安卓设备上使用 BlurLyric 音乐播放器的用户。

## 目标
- 使 BlurLyric 应用能够在安卓平台上成功构建和运行
- 保持与桌面版本一致的核心功能和用户界面
- 适配安卓设备的屏幕尺寸和交互方式
- 确保应用在安卓平台上的性能和稳定性

## 非目标（范围外）
- 不修改应用的核心功能逻辑
- 不添加仅针对安卓平台的新功能
- 不涉及其他移动平台（如 iOS）的适配
- 不改变现有的桌面平台支持

## 背景与上下文
- BlurLyric 是一个基于 Vue 3 和 Tauri 开发的音乐播放器应用
- 当前版本仅支持桌面平台（Windows、macOS、Linux）
- Tauri 2.0+ 版本已支持安卓平台，为跨平台开发提供了可能
- 应用使用了 Tauri 插件如 fs 和 shell，需要确保在安卓平台上的兼容性

## 功能需求
- **FR-1**: 应用能够在安卓平台上成功构建和安装
- **FR-2**: 保持与桌面版本一致的音乐播放核心功能
- **FR-3**: 适配安卓设备的屏幕布局和交互方式
- **FR-4**: 确保文件系统访问和音乐库扫描功能在安卓平台上正常工作
- **FR-5**: 保持应用的视觉设计和用户体验一致性

## 非功能需求
- **NFR-1**: 应用在安卓平台上的启动时间不超过 3 秒
- **NFR-2**: 音乐播放过程中 CPU 使用率不超过 20%
- **NFR-3**: 应用在安卓设备上的内存占用不超过 200MB
- **NFR-4**: 支持 Android 7.0 (API 24) 及以上版本
- **NFR-5**: 应用界面在不同屏幕尺寸的安卓设备上都能正常显示

## 约束
- **技术**: 基于 Tauri 2.0+ 框架，使用 Rust 和 Vue 3
- **依赖**: 依赖 Tauri 提供的安卓平台支持，以及相关插件的安卓兼容性
- **性能**: 需要考虑安卓设备的性能差异，确保在中低端设备上也能流畅运行

## 假设
- Tauri 2.0+ 框架的安卓平台支持已经稳定可用
- 应用的核心功能逻辑在安卓平台上不需要重大修改
- 应用的资源文件和图标已经适用于安卓平台

## 验收标准

### AC-1: 安卓平台构建成功
- **Given**: 开发环境已配置完成，Tauri 安卓依赖已安装
- **When**: 执行 Tauri 安卓构建命令
- **Then**: 构建过程完成，生成可安装的 APK 文件
- **Verification**: `programmatic`
- **Notes**: 构建过程不应出现错误，APK 文件应能正常安装到安卓设备

### AC-2: 核心功能正常运行
- **Given**: 应用已安装到安卓设备
- **When**: 打开应用并测试音乐播放、库扫描、歌词显示等核心功能
- **Then**: 所有核心功能与桌面版本表现一致
- **Verification**: `programmatic`
- **Notes**: 功能测试应覆盖音乐播放、暂停、上一曲、下一曲、音量调节、歌词显示等基本操作

### AC-3: 界面适配良好
- **Given**: 应用在不同屏幕尺寸的安卓设备上运行
- **When**: 查看应用界面布局和交互元素
- **Then**: 界面布局自适应不同屏幕尺寸，交互元素大小适合触摸操作
- **Verification**: `human-judgment`
- **Notes**: 应测试至少三种不同屏幕尺寸的设备（小屏手机、大屏手机、平板）

### AC-4: 文件系统访问正常
- **Given**: 应用已安装到安卓设备
- **When**: 扫描本地音乐库或访问本地文件
- **Then**: 应用能够正确访问和扫描本地音乐文件
- **Verification**: `programmatic`
- **Notes**: 需要测试应用是否能够获取存储权限并正确扫描音乐文件

### AC-5: 性能符合要求
- **Given**: 应用在安卓设备上运行
- **When**: 监测应用的启动时间、CPU 使用率和内存占用
- **Then**: 启动时间不超过 3 秒，CPU 使用率不超过 20%，内存占用不超过 200MB
- **Verification**: `programmatic`
- **Notes**: 性能测试应在中低端安卓设备上进行

## 待解决问题

- [ ] Tauri 插件（如 fs 和 shell）在安卓平台上的兼容性如何
- [ ] 应用的窗口配置如何适配安卓平台的活动（Activity）概念
- [ ] 如何处理安卓平台的屏幕旋转和生命周期事件
