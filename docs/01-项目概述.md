# 01 - 项目概述

## 1.1 项目基本信息

| 项目属性 | 内容 |
|----------|------|
| 项目名称 | BlurLyric-3.0 |
| 项目类型 | 跨平台桌面音乐播放器应用 |
| 当前版本 | 3.0.0 |
| 开发框架 | Tauri 2.0 + Vue.js |
| 目标平台 | Windows / macOS / Linux |
| 开发者 | gozaoo |

## 1.2 项目愿景

构建一个高性能、现代化的音乐播放器，具备以下核心能力：

1. **本地音乐管理** - 高效扫描、解析和管理本地音乐库
2. **多源音乐整合** - 支持本地文件、WebDAV、网络API等多种音乐来源
3. **智能缓存策略** - 多级缓存机制，优化资源访问性能
4. **用户偏好系统** - 歌单管理、收藏功能、播放历史
5. **跨平台一致性** - 统一的用户体验和功能特性
